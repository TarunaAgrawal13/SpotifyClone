Spotify Clone UI 🎵

A simple and visually appealing Spotify Clone UI built to practice frontend development and modern UI design.
This project recreates the look and feel of Spotify with a clean music player interface, sidebar navigation, and responsive layout.

Features
🎧 Spotify-inspired user interface
📂 Sidebar navigation menu
🎵 Music cards and playlists section
💻 Responsive and modern design
✨ Smooth and visually appealing layout
🎨 Clean styling and user-friendly interface
Tech Stack
HTML
CSS
JavaScript
